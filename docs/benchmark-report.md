## 二、各模型詳細數據

### 2.1 Qwen3-8B（主力推薦）

```
TPS:       13.4 / 13.4 / 13.5（5 次測試極穩定）
延遲:       3.71 - 3.74s（極穩定）
記憶體:     ~90 GB（模型 + KV cache）
輸出長度:   200 tokens / 次
vLLM 相容: ✅ 完全支援
```

**適合場景**：SQL 生成、pandas 分析、RAG 檢索、HybridRAG、工具呼叫
**是否推薦**：✅ **是，主力模型**

### 2.2 Qwen3-0.6B（備用降載）

```
TPS:       105.7 / 109.6 / 110.1（最快）
延遲:       ~0.26s
記憶體:     ~2 GB
輸出長度:   200 tokens / 次
vLLM 相容: ✅ 完全支援
```

**適合場景**：簡單問答、降載備用、低資源環境
**是否推薦**：✅ 是

### 2.3 Qwen2.5-0.5B（測試用）

```
TPS:       151 TPS（最快）
延遲:       0.04s
記憶體:     ~1 GB
```

**適合場景**：功能驗證、開發測試
**是否推薦**：⚠️ 僅供開發測試

### 2.4 Qwythos-9B（不建議）

```
載入結果：❌ vLLM 引擎初始化失敗
原因：     Gated DeltaNet 線性注意力架構需要特殊 kernel
狀態：     下載完成但無法穩定載入
是否推薦： ❌ 不建議
```
